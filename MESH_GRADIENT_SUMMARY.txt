================================================================================
KEY CONCEPTS EXPLAINED
================================================================================

1. BICUBIC HERMITE PATCH (BHP) MESH
   - Smooth surface interpolation using cubic Hermite curves
   - Defined by 4 control points and tangent vectors
   - Formula: S(u,v) = H(u) * M * H(v)^T
   - Results in smooth, organic deformations

2. CONTROL POINT SYSTEM
   - 2D grid of control points (typically 3×3 to 6×6)
   - Each point has:
     * Location (x, y) in normalized device coordinates [-1, 1]
     * Color (r, g, b) for per-vertex coloring
     * U-tangent and V-tangent vectors for curve control
     * Rotation angles (uRot, vRot) and scale factors (uScale, vScale)

3. MESH SUBDIVISION
   - Each patch between 4 control points is subdivided
   - Default: 50×50 subdivisions per patch
   - Results in 10,000+ vertices for smooth rendering
   - Performance-critical: uses matrix pre-computation

4. ANIMATION SYSTEM
   - Time-based flow: frameTime += frameDelta * flowSpeed
   - Audio-responsive: smoothedVolume smooths input with ~100ms response
   - State transitions: alpha-based crossfading between mesh states
   - Shader effects: UV rotation, volume dampening, vignetting

5. IMAGE PROCESSING
   - Album art → 32×32 texture (extreme size reduction)
   - Applied transforms:
     * Contrast: 0.4 → 1.7 (increases vibrancy)
     * Saturation: 3.0 (boosts color)
     * Brightness: 0.75 (darkens slightly)
     * Gaussian blur: radius 2 (smooths for sampling)

================================================================================
CRITICAL FORMULAS
================================================================================

Hermite Basis Functions:
  h₀(t) = 2t³ - 3t² + 1           (interpolates to p₀ at t=0)
  h₁(t) = -2t³ + 3t²              (interpolates to p₁ at t=1)
  h₂(t) = t³ - 2t² + t            (interpolates tangent at p₀)
  h₃(t) = t³ - t²                 (interpolates tangent at p₁)

Hermite Matrix H (basis transformation):
  [  2  -2   1   1 ]
  [ -3   3  -2  -1 ]
  [  0   0   1   0 ]
  [  1   0   0   0 ]

Surface Evaluation:
  P(u,v) = [u³ u² u 1] * H^T * M_pos * H * [v³ v² v 1]^T

Tangent Vector Computation:
  uTangent = [cos(uRot) * uScale, sin(uRot) * uScale]
  vTangent = [-sin(vRot) * vScale, cos(vRot) * vScale]

Volume Smoothing (Low-Pass Filter):
  smoothed += (target - smoothed) * min(1.0, Δt / 100ms)

Color Adjustments:
  contrast(x) = (x - 128) * factor + 128
  saturation(x) = lerp(gray, x, factor)
  brightness(x) = x * factor

Memory Usage Estimate:
  - Vertex data: 10,000 vertices × 7 floats × 4 bytes = 280 KB
  - Index data: 6,000 indices × 2 bytes = 12 KB
  - Texture: 32×32 RGBA = 4 KB
  - Total: ~300 KB per mesh state

Optimization Strategies Employed:
  1. Matrix pre-computation: Acc = H^T * M * H
  2. Power value caching: Pre-compute u³, u², u for all subdivisions
  3. Batch vertex updates: Single setVertexData() call
  4. Framebuffer object compositing: Correct alpha blending
  5. Adaptive frame skipping: Maintains target FPS
  6. Static mode: 0% CPU when animation paused

================================================================================
RENDERING PIPELINE
================================================================================

Frame Rendering Sequence:
  1. RequestAnimationFrame callback triggered
  2. Check FPS interval (skip if too soon)
  3. Update frameTime with flowSpeed multiplier
  4. Update mesh state alphas (fade in/out transitions)
  5. Check canvas resize (update viewport if needed)
  6. For each MeshState:
     a. Render mesh to Framebuffer Object (FBO)
        - Set shader uniforms (time, volume, alpha)
        - Bind mesh vertices and texture
        - Draw triangles
     b. Composite FBO to screen
        - Enable alpha blending
        - Draw full-screen quad with FBO texture
     c. Apply easing: easeInOutSine(alpha)
  7. gl.flush() to ensure completion
  8. Schedule next frame

Shader Effects Pipeline (Fragment Shader):
  1. Dithering: Add 1/255 gradient noise (reduce banding)
  2. UV Rotation: Rotate around center, scaled by volume
  3. Texture Sampling: Sample rotated UV from album texture
  4. Color Modulation: Multiply by vertex color and alpha
  5. Vignetting: Darken edges using distance from center
  6. Alpha Blending: SRC_ALPHA, ONE_MINUS_SRC_ALPHA
